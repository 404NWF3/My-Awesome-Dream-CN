# 预设提示词清单

预设提示词是 grill-me-study 深挖到对应分支时**可调用的弹药库**：调出对应提示词让用户作答，回复沉淀进 `wiki/questions/`（`prompt_type: preset`、`preset_slug` 对应本页条目），被展示面板第 4 区命中。

每条条目映射到 grill-me-study 决策树的一个**关键节点**（形态 A relentless 深挖，两路径 Q3/Q4 合流，R2 沉淀）。

## 预设提示词条目

### 决策树根节点

- **preset_slug**: `competitiveness-self`
  - **适用于**: Q0——保研竞争力自评
  - **原文**: "根据你的本科院校、绩点排名、科研竞赛经历、英语水平，你目前的保研竞争力处于什么水平？请分项盘点：1. 绩点/排名 2. 科研/论文 3. 竞赛/获奖 4. 英语/证书 5. 推荐信/关系 6. 软实力。"

### 路径甲（中/低竞争力 → 挖动机）

- **preset_slug**: `goal-clarify`
  - **适用于**: Q1——为什么想读研
  - **原文**: "请诚实地回答：你选择保研的根本原因是什么？是学术追求（想继续做研究）、就业出路（目标行业需要硕士学位）、家庭期待（家长希望你读研），还是主要是惯性（身边人都在保研，不走这条路心里不踏实）？不要给我'正确答案'，给我你的真实想法。"

- **preset_slug**: `research-direction`
  - **适用于**: Q2（学术追求分支）——研究兴趣是否真实
  - **原文**: "你说你想做学术。请具体说出：你想研究什么方向？这个方向的核心问题是什么？你有没有读过该方向近三年的顶会/顶刊论文？你能否举出一篇你印象最深的论文并告诉我为什么它让你兴奋？如果你现在说不出，我们就回到上面重新问——你想读研是不是因为其他原因。"

- **preset_slug**: `employment-gateway`
  - **适用于**: Q2（就业出路分支）——读研是跳板还是门槛
  - **原文**: "你说读研是为了就业。请区分：你目标行业/岗位是真有硕士硬门槛（如金融行业某些岗位、高校教职），还是硕士只是'加分项'？如果是后者——同等时间投入去工作攒经验、和去读个硕士再找工作，哪个 ROI 更高？请给出你的判断而非猜测。"

- **preset_slug**: `habit-vs-motive`
  - **适用于**: Q2（惯性/还没想清楚分支）——对撞真实动机
  - **原文**: "你刚才说你还没想清楚。那我们先换个角度：如果不读研你会去干什么？你能接受那个结果吗？你身边劝你读研的人——家人、同学、辅导员——他们各自的理由是什么？这些理由是你认同的，还是你只是不想让他们失望？把'他们的期待'和'你自己的意愿'分开说清楚。"

### 路径乙（高竞争力 → 盘牌）

- **preset_slug**: `hand-audit`
  - **适用于**: Q1（高竞争力路径）——手上有什么牌
  - **原文**: "你的竞争力自评偏高——我们来认真盘点一下你的牌。请在以下维度逐一打出'强/中/弱'并给出一行证据：绩点排名 / 科研经历（有无论文/项目）/ 竞赛获奖 / 英语成绩 / 推荐信来源 / 实习或实践。打完牌后回答：这副牌，最强的一张是什么？最弱的一张是什么？哪张牌你觉得可以继续加强但时间不够了？"

- **preset_slug**: `hand-direction`
  - **适用于**: Q2（高竞争力路径）——牌打向哪
  - **原文**: "你手上的牌不错。现在决定方向：这副牌打向学术（直博/学硕做研究）还是打向就业（专硕刷学历进大厂/金融机构）？不要同时选两端。你的牌面、你过去的精力分配、你对五年后的想象——哪一个方向更占上风？"

### 合流后

- **preset_slug**: `willing-cost`
  - **适用于**: Q3——愿付什么代价
  - **原文**: "保研有代价。请诚实地回答你愿意承担以下代价到什么程度：1. 多读 1–2 年（专硕 vs 学硕 vs 直博的时间账）2. 异地（你愿意去哪个城市？不愿意去哪个？），3. 降院校层（你最差能接受什么学校？），4. 转专业/方向的不确定性。把'绝对不能接受'的那条圈出来。"

- **preset_slug**: `degree-target`
  - **适用于**: Q4——目标学位浮现
  - **原文**: "基于你的动机和愿付代价，你的目标学位已经开始清晰了。学硕、专硕、直博——你的选择是什么？为什么是这个不是另外两个？如果你选专硕：你清楚专硕的学费/住宿/就业差别吗？如果你选直博：你做过'读不下去退学'的心理预案吗？给出你的决策和一句理由。"

- **preset_slug**: `geo-city`
  - **适用于**: Q5——地域偏好与城市约束
  - **原文**: "地域对你有多重要？请按优先级排序：1. 留在某个特定城市（为什么？家庭/对象/房价？），2. 只去一线城市，3. 不限地域哪里好去去哪里。如果你选了 1，是否有'强制约束'——比如父母需要照顾、伴侣在同一城市工作、你已经在该城市买了房？如果是强制约束，列出来，我们后面的院校筛选会自动过滤。"

- **preset_slug**: `target-match`
  - **适用于**: Q6——目标院校层匹配
  - **原文**: "结合你的画像（绩点、科研、竞赛、动机、代价、地域），以下院校层级中哪个是你的主攻方向：顶尖 C9 联盟 / 中流 985 / 211 / 本校保底？请给出你的理想校、主攻校、保底校各 1–3 所。对于最想去的那个院校/学院/项目：你具体了解它的夏令营门槛吗？如果现在还不知道，这就是下一步该查的。"

- **preset_slug**: `timeline-check`
  - **适用于**: Q7——时间线倒推
  - **原文**: "现在到推免系统开放还有多少时间？按月倒推：你需要完成哪些动作——1. 补齐绩点/科研/竞赛短板 2. 准备文书（个人陈述/简历/推荐信），3. 投递夏令营/预推免，4. 面试准备。从最后截止日开始逐一倒排到本月底，列出每个月的关键里程碑。如果某个节点和课业/考试撞车，标出来。"

- **preset_slug**: `swot-gap`
  - **适用于**: Q8——SWOT + 差距分析
  - **原文**: "根据你选定的目标院校层，请对比你的当前条件与该院校夏令营/预推免的往年要求：1. 绩点/排名差多少？2. 科研/竞赛差什么？3. 英语/证书差什么？4. 推荐信去哪弄？每项差距给出一个'弥补计划'——谁能帮你、多长时间、做到什么程度。差距大到无法弥补的——这就是备选方案启动的信号。"

### 输出

- **preset_slug**: `direction-card`
  - **适用于**: 终——保研方向卡片
  - **原文**: "汇总以上全部——动机、牌面、代价、目标学位、目标院校层、地域、时间线、SWOT——产出一张'保研方向卡片'。定性结论写进你的个人画像 entity（学硕/专硕、目标院校层、动机），规划内容（时间线/材料清单/风险预案）写进 wiki/strategy/我的保研方向.md。"

## 注

- preset_slug 一旦录入就**不再变更**（沉淀文件靠它回链），新增走新 slug。
- 提示词原文更新 = 新 slug（旧条目保留，老沉淀仍能命中）。
- 共计 12 条，覆盖决策树全路径（根节点 → 路径甲/乙 → 合流 → 输出）。grill-me-study 自行判断当前处于哪个节点、调用对应条目。
